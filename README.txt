Los Santos Police Department — static website package

Что внутри:
- index.html
- styles.css
- script.js
- assets/logo.webp
- assets/hero.png

Как опубликовать:
1. Распакуйте ZIP.
2. Загрузите все файлы на хостинг или перетащите папку на Netlify Drop.
3. Убедитесь, что index.html находится в корне сайта.
4. Откройте полученную ссылку — сайт будет доступен публично.
